# PPC-Invorio-mappa
Mappa interattiva del Piano di Protezione Civile del comune di Invorio.
All'apertura della mappa tutti i rischi sono attivi, puoi selezionare solo quelli di tuo interesse in alto a destra.
Sulla sinistra puoi modificare lo zoom o cercare l'indirizzo che preferisci tramite la lente di ingrandimento.
